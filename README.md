# Bubble Sort 

The goal of this project was to learn about the basic Ruby syntax and to learn about the yield method used by ruby. 

## Output

```ruby
input
["hi", "hello", "hey"]
[4, 3, 78, 2, 0, 2]

output
["hi", "hey", "hello"]
[0, 2, 2, 3, 4, 78]
```
